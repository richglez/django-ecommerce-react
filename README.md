# Django-React Ecommerce
This is a full stack e-commerce project that uses Django Rest Framework for the backend and React with TypeScript for the frontend, all packaged with Vite.


## Description
It's an online marketplace with a wide variety of products, from clothing to electronics, all in one place.

* Public Store: Catalog with filters, shopping cart, and checkout
* User Panel: Profile, order history ("My Orders"), and shipping addresses
* Administration Panel: Product, user, and order management
